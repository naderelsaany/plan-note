# 📐 معايير التصميم — Professional Dev Skills

## اختيار Design System

| لو المستخدم عايز | اختار من popular-web-designs |
|-----------------|------------------------------|
| موقع شخصي / CV | Stripe أو Vercel أو Linear |
| Dashboard / أداة | Linear, Vercel, Sentry, Supabase |
| SaaS / منصة | Stripe, Linear, Notion |
| متجر | Shopify, Stripe |
| AI / LLM | OpenAI, Claude, Perplexity, Cursor |
| Entertainment | Spotify, Apple |
| مقال / Blog | Mintlify, Sanity, Notion |
| شركة / Enterprise | IBM, Vercel, Stripe |
| عربي / RTL | Stripe (يدعم RTL) أو من 152 Design System |

## قواعد أساسية لأي Design System

1. **Dark mode أولاً** — #0a0e17 أو #08090a مش #000 أبداً
2. **Accent واحد أساسي** — لون واحد 90% من الوقت
3. **Borders شفافة** — gba(255,255,255,0.08) داكن، gba(0,0,0,0.08) فاتح
4. **Multi-layer shadows** — دايم 2-3 layers
5. **Letter spacing سالب للهيدرات** — كل ما الهيدر أكبر، tracking أضيق
6. **تجنب الأسود الخالص** #000 — استخدم #0f0f0f
7. **تجنب الأبيض الخالص** #fff — استخدم #f7f8f8
8. **Contrast:** body text ≥4.5:1
