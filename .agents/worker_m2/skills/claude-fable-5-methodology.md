# 🎯 Fable 5 Methodology — الدروس المستفادة

## 5 Core Mechanisms

| # | الآلية | التطبيق عندي |
|---|--------|--------------|
| 1 | **Adaptive Extended Thinking** | على المهام المعقدة، أبطئ وفكر قبل أول action |
| 2 | **Agent Harness Architecture** | Plan → Execute → Verify → Correct → Iterate |
| 3 | **Self-Verification** | اختبر شغلك قبل ما تبلغ. اكتب test، شغّل الكود |
| 4 | **Persistent Memory** | استخدم memory/skills للمهام الطويلة |
| 5 | **Context Engineering** | System prompt ثابت + تفاصيل متغيرة في الآخر |

## Lessons Applied

1. **Slow down on complex tasks** — think before acting
2. **Test my own output** — run code, verify results
3. **Use memory actively** — save procedures as skills
4. **Batch context upfront** — read all needed files at once
5. **Use delegate_task** for parallel workstreams
6. **Save reusable knowledge** as skills
