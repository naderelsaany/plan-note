# 🧠 ذاكرة المهندس (Agent Memory & Checklist)
> **الهدف:** تتبع المهام بدقة، المراجعة الذاتية، والتأكد من عدم نسيان أي قاعدة من قواعد البرومبت.

## القواعد الصارمة (للمراجعة قبل دمج أي فرع)
- [ ] هل تم التنفيذ في فرع (Branch) مستقل؟
- [ ] هل تم الرفع كـ Vercel Preview؟
- [ ] هل تم عمل اختبار Regression لـ (تسجيل الدخول، Dashboard، حفظ الملاحظات، حفظ اللوحة)؟
- [ ] هل اللوحات القديمة تفتح بشكل سليم (Backward Compatibility)؟
- [ ] هل طلبنا موافقة المالك قبل دمج الفرع في `main`؟
